Using ICD-[X] Codes with FHIR

[%wgt vocab%] Work GroupMaturity Level: N/ABallot Status: STU 3

ICD is a family of code systems maintained by WHO, with many countries publishing their own variants.

The are two principal versions of ICD in use - ICD-9 and ICD-10 (note that while US usage has recently updated to ICD-10, there is still a lot of legacy data coded in ICD-9). Though these are regarded as different versions of ICD, they redefine the meaning of codes, and so are regarded as entirely separate code systems.

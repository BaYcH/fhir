
=== Sheet Bindings =====================================

 -- Row 1 -----------------------------------
  Binding Name = ContraindicationCategory
  Binding = value set
  Reference = valueset-contraindication-category
  Definition = Codes identifying the general type of contraindication.  E.g. Drug-drug interaction, Timing issue, Duplicate therapy, etc.
  Conformance = preferred

 -- Row 2 -----------------------------------
  Binding Name = ContraindicationMitigationAction
  Binding = value set
  Reference = valueset-contraindication-mitigation-action
  Definition = Codes describing steps taken to resolve the contraindication or other circumstances that mitigate the risk associated with the contraindication.  E.g. \'added concurrent therapy\', \'prior therapy documented\', etc.
  Conformance = preferred

 -- Row 3 -----------------------------------
  Binding Name = ContraindicationSeverity
  Binding = code list
  Reference = #contraindication-severity
  Definition = Indicates the potential degree of impact of the identified issue on the patient
  Conformance = required
  v3 = http://hl7.org/fhir/ValueSet/v3-SeverityObservation
